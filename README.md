# Vali-Product
Manual for starting the app,
Server side: 
1. Change directory by using "cd api" command
2. In commandline type "npm install"
3. After installation is finished type "nodemon server"

Client side:
1.  Change directory by using "cd client" command
2. In commandline type "npm install"
3. After installation is finished type "npm start"
